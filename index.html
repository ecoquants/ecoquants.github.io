Services

Analyze

We use the latest, most efficient data science techniques from ingestion and preparation of data to modeling, visualization and online communication. We specialize in R and Python scientific programming languges, with an emphasis on reproducible reearch (eg with Rmarkdown, git, Github).

Develop

We rapidly develop interactive data-driven web applications, particularly those with complexity in space, time and taxa.
